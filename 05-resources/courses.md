## Free Courses

### 1. DeepLearning.AI: AI Agentic Design Patterns
- **Platform**: DeepLearning.AI (free)
- **Instructor**: Andrew Ng
- **What you'll learn**: Agent design patterns, tool use, planning
- **Link**: [deeplearning.ai](https://deeplearning.ai)

### 2. LangChain Academy
- **Platform**: LangChain (free)
- **What you'll learn**: LangGraph, state management, deployment
- **Link**: [academy.langchain.com](https://academy.langchain.com)

### 3. CrewAI Documentation Tutorials
- **Platform**: CrewAI (free)
- **What you'll learn**: CrewAI framework, multi-agent systems
- **Link**: [docs.crewai.com](https://docs.crewai.com)

### 4. AutoGen Tutorials
- **Platform**: Microsoft / AG2 (free)
- **What you'll learn**: Conversational agents, group chat
- **Link**: [ag2.ai](https://ag2.ai)

### 5. Hugging Face Agents Course
- **Platform**: Hugging Face (free)
- **What you'll learn**: Agent fundamentals, tool use
- **Link**: [huggingface.co/learn](https://huggingface.co/learn)

### 6. Google's Agent Development Guide
- **Platform**: Google (free)
- **What you'll learn**: ADK, Vertex AI agents
- **Link**: [cloud.google.com](https://cloud.google.com)

### 7. Fast.ai: Practical Deep Learning
- **Platform**: Fast.ai (free)
- **What you'll learn**: LLMs, fine-tuning, applications
- **Link**: [fast.ai](https://fast.ai)
